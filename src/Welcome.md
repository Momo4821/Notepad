# Welcome

## Welcome to Notepad! This is a simple text editor that allows you to create and edit plain text files. Whether you're jotting down notes, writing code, or drafting documents, Notepad provides a straightforward and user-friendly interface.


## Purpose
 - I wanted to create a simple text editor to improve my coding skills and start my journey in software development. This project serves as a learning experience and a way to practice programming concepts.





## Technologies Used
- **Programming Language:** C#
- **Framework:** .NET
- **GUI Library:** Windows Presentation Foundation(WPF)
- **Development Environment:** Visual Studio, Jetbrains Rider
- **Version Control:** Git, GitHub
- **Build Tools:** MSBuild



## Features
- **Basic Text Editing:** Create, open, edit, and save plain text files.
- **User-Friendly Interface:** Simple and intuitive design for easy navigation.
- **Cross-Platform Compatibility:** Runs on Windows
